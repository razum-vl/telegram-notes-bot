# 📝 Telegram Bot для заметок

Простой и удобный Telegram бот для создания и управления личными заметками. Написан на Go с использованием in-memory хранилища.

## ✨ Возможности

- ✅ **Создание заметок** - сохраняйте важную информацию
- 📋 **Просмотр списка** - все заметки в одном месте
- 🔍 **Получение по заголовку** - быстрый доступ к нужной заметке
- 🗑️ **Удаление заметок** - управляйте своей коллекцией
- 🚀 **Мгновенная работа** - все заметки в памяти для быстрого доступа
- 🔒 **Безопасно** - каждый пользователь видит только свои заметки

## 🛠️ Технологии

- **Язык:** Go 1.21+
- **Библиотеки:**
  - [go-telegram-bot-api](https://github.com/go-telegram-bot-api/telegram-bot-api) - работа с Telegram API
  - [godotenv](https://github.com/joho/godotenv) - загрузка переменных окружения
- **Хранилище:** In-memory с конкурентным доступом (sync.RWMutex)

## 📋 Команды бота

| Команда | Описание | Пример |
|---------|----------|--------|
| `/start` | Приветствие и список команд | `/start` |
| `/save` | Сохранить заметку | `/save Покупки; Купить молоко` |
| `/get` | Получить заметку по заголовку | `/get Покупки` |
| `/list` | Показать все заметки | `/list` |
| `/delete` | Удалить заметку | `/delete Покупки` |

## 🚀 Быстрый старт

### 1. Получите токен бота

1. Найдите в Telegram [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и сохраните полученный токен

### 2. Установите и запустите

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/notes-telegram-bot.git
cd notes-telegram-bot

# Создайте файл с токеном
echo "TELEGRAM_BOT_TOKEN=ваш_токен_здесь" > .env

# Запустите бота
go run main.go
